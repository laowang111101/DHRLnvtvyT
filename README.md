# 物资管理系统简介 java716

## 项目概述

物资管理系统是一款基于 Spring Boot 和 Vue 的前后端分离项目，旨在为企业和仓库提供物资管理解决方案。该项目分为管理员端和用户端，两端功能各异，以适应不同角色的操作需求。

## 技术栈

- 后端：Spring Boot
- 前端：Vue
- 开发工具：VSCode、IntelliJ IDEA

## 功能模块

### 管理员端

- 通知管理
- 用户管理
- 部门管理
- 权限管理
- 采购管理
- 申领管理
- 数据分析

### 用户端

- 通知管理
- 申领
- 采购
- 仓库管理

## 系统角色

- 管理员：负责系统管理、通知发布、用户管理等操作
- 用户：负责申领、采购、仓库管理等操作

## 运行环境

- 后端：Java 1.8+，MySQL 5.7+
- 前端：Node.js 12.0+，现代浏览器
